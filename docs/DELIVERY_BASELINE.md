# 当前交付基线

## 1. 文档用途

本文件用于说明当前版本的可交付范围、运行方式和未完成项，作为开发版交付说明。

## 2. 当前可交付内容

### 2.1 账号与身份

- 注册、登录、登出
- 个人资料更新
- 昵称、用户名、域名身份卡与资料可见范围
- 用户搜索与公开资料查看

### 2.2 空间与订阅

- 空间统一入口下的私人/公共空间展示与创建
- 会员等级展示
- 订阅创建与当前订阅查看

### 2.3 社交内容

- 发布、查看、编辑文章
- 点赞、评论、转发
- 作者主页与个人内容流

### 2.4 聊天

- 会话列表
- 会话详情
- 消息发送
- 未读统计与排序规则
- 文本、图片、视频、语音消息发送
- 媒体消息压缩与过期自动删除
- 新好友浮动提醒与新消息角标提示

### 2.5 区块链账号扩展

- 外部账号绑定
- 外部账号解绑
- 基础地址格式与签名载荷校验
- 主页链上账号摘要展示

## 3. 当前运行方式

- 账号服务：`make run-account`
- 空间服务：`make run-space`
- 通讯服务：`make run-message`
- 版本化迁移：`make migrate`
- 基础检查：`make test`
- 后端构建：`make build`
- 本地冒烟：`make smoke`

## 4. 当前交付产物

- 后端源码
- 前端静态页面源码
- 规划与设计文档
- API 文档与数据模型文档
- 本地构建产物：
  - `build/account-service`
  - `build/space-service`
  - `build/message-service`
  - `build/migrate`

## 5. 当前未覆盖项

- 正式生产部署脚本
- 容器化交付方案
- 前端正式打包发布方案
- 区块链真实签名验签

## 6. 适用结论

- 当前版本适合作为开发版、联调版和内部演示版交付
- 若进入生产交付，建议先补齐部署自动化和真实验签流程
